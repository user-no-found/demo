# 变更提案：自我更新功能

## 概述
为程序升级器添加自我更新功能，当源目录中存在与程序本身同名的文件时，能够通过临时脚本/子进程完成自身替换。

## 需求场景
### 场景 1：程序自我更新
- Given 源目录中存在与程序本身同名的可执行文件
- When 程序检测到需要更新自己
- Then 生成临时脚本，由脚本完成替换操作，然后启动新版本程序

## 技术方案
采用临时脚本方案：
1. 程序检测到源目录中有与自身同名的文件
2. 生成临时批处理脚本（Windows下为.bat文件）
3. 脚本内容：
   - 等待当前程序退出
   - 复制新版本到程序所在目录
   - 启动新版本程序
   - 删除临时脚本自身
4. 启动脚本后程序退出
5. 脚本执行完成后，新版本程序启动

## 影响范围
- updater.rs：添加自我更新检测和脚本生成逻辑
- main.rs：修改主流程，在普通更新前先检测自我更新
